# Codebook

## Purpose
This Code book will be used to describe variables in Averages.txt.

## Identifier variables
* Activity Name

Indicates the activity performed by the participant.
There were 6 different activities in the original study:
        + WALKING
        + WALKING_UPSTAIRS
        + WALKING_DOWNSTAIRS
        + SITTING
        + STANDING
        + LAYING
* Subject ID

Indicates the Participant ID. Each ID is in numerical form. Each distinct ID indicates each distinct participant.
